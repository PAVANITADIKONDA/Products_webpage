# 🛍️ Products Web Application

## 📌 Project Overview
This is a simple **Products Web Application** developed using **HTML, CSS, and JavaScript**.  
The application allows users to register, log in, view products, and see detailed information about each product.

## 🚀 Features
- User Registration
- User Login
- Home Page
- Product Cards Display
- View Product Details
- Simple and clean UI

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## ▶️ How to Run the Project
1. Download or clone the repository.
2. Open the project folder.
3. Run **Home.html** in your browser.
